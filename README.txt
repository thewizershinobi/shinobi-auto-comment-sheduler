Shinobi Auto Comment Scheduler automatically posts relevant comments on existing WordPress posts at scheduled intervals, increasing engagement and keeping discussions active.

== Description ==
Shinobi Auto Comment Scheduler is a powerful WordPress plugin that automatically posts relevant comments on existing posts based on a user-defined schedule. Ideal for increasing engagement and keeping your content fresh.

**Features:**
- Schedule automatic comments on existing posts.
- Customizable comment templates and usernames.
- Manual comment trigger option.
- Notifications for exhausted usernames or comments.
- Secure and efficient WordPress integration.

== Installation ==
1. Upload the `shinobi-auto-comment-scheduler` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to Settings > Shinobi Auto Comment Scheduler to configure.

== Frequently Asked Questions ==

= How does the scheduling work? =
The plugin uses WordPress cron jobs to schedule and post comments automatically.

= Can I manually trigger a comment? =
Yes, a manual trigger button is available in the admin settings.

= Is this plugin secure? =
Yes, the plugin follows WordPress security best practices, including nonce verification and data sanitization.
